# vine-lang

vine-lang 是基于 Go 语言实现的 [Vine programming language](https://github.com/xiaoxustudio/vine-lang).

# 安装

下载`Release`界面的`vine-lang`二进制文件

放置到任意目录即可使用。

推荐添加到`PATH`中，方便使用。

# 使用

```shell
vine examples\001.vine
```
